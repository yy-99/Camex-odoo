📦 Camex Delivery Integration

Seamless shipping and tracking directly from Odoo

✅ Features

  • 🔄 Auto-generate shipments upon order confirmation
  • 🛠️ API configuration panel inside Odoo
  • 🗺️ Updates delivery status automatically in Odoo

🛠️ How to Configure

  • To define your cities and areas, go to Contacts → Configuration, then set up your own Base Cities and Base Areas.
Create a new Delivery Method, select the provider "CAMEX", then fill in the fields in the order shown in the image.
By navigating to Contacts → CAMEX Mapping, create a mapping between the areas you previously defined and the CAMEX areas.
